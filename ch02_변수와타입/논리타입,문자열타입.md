# Chapter 02 변수와 타입
## 논리 타입
참과 거짓을 의미하는 논리 리터럴은 true와 false. 논리 리터럴은 boolean 타입 변수에 대입 가능.
```
boolean stop = true;
boolean stop2 = false;
```

boolean 타입 변수는 주로 두 가지 상태값 저장할때 사용. 상태값에 따라 조건문과 제어문의 실행 흐름을 변경.
```
boolean stop = true;
if(stop) {
  System.out.println("중지합니다.");
} else{
  System.out.println("시작합니다.");
}
```
